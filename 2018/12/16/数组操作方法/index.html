数组操作方法

December 16 2018

数组的检测：1.if ( value instanceof Array){ //操作 }

2.Array.isArray( value )

3.利用数组的特性，例如split(),substr()方法

toLocaleString()方法：把数组转化为本地字符串

join()方法：以参数为间隔将数组连接起来

栈方法：push()方法：添加一个项到数组的末尾 pop()方法：去掉数组最后一项，并返回那个项

队列方法：unshift()方法：在数组首添加一项， shift()方法：在数组首去掉一项，并返回

重排序方法：.reverse()反转数组项 sort()：按升序排列数组项（非从小到大）,从小到大方法： 操作方法：slice（）方法：基于的当前数组的一项或多项创建一个新数组，参数为始末下标，（不包括末尾下标）

splice（）方法：用于删除，插入或替换,参数一：要删除的第一项的位置， concat（）方法：用于连接两个或多个数组

split()方法：用于把一个字符串分割成字符串数组（以参数一为间隔，参数二为规定长度）

substr（start，length）方法:用于抽取字符串

位置方法：indexOf(要查找的值),lastINdexOf(),返回查找到的下标，否则-1.
