typeof

typeof用于基本类型的判断，可取的值有七个：

  1. string
  2. number
  3. boolean
  4. object
  5. symbol
  6. function
  7. undefined

<font color="red">注意：typeof null == Object //true</font> 相关原理： JS在底层如何实现类型信息呢？在存储变量的机器码的低1~3位进行区分：

  • 000：对象
  • 010：浮点数
  • 100：字符串
  • 110：布尔
  • 1：整数 特殊：
  • null：均为0
  • undefined: 用-2^30来表示

所以，在判断null类型时，由于全是0被当作对象来判断
