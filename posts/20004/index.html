居中

居中根据条件适用分为定宽高和不定宽高
定宽高：

  • absolute + margin(取值auto)
  • absolute + margin(值为宽度的负50%)
    不定宽高：
  • text-align(水平居中) + vertical-align(垂直居中)
  • absolute + transform
  • flex
  • gird

三栏布局

实现左右固定，中间自适应：

  • float
  • 双飞翼
  • flex

float

左右分别往左右浮动，中间设置margin-left/right

双飞翼

中间内容左移并占满，左内容float:left并margin-left:-100%(左边框和中间内容左边框对齐),右内容float:left并margin-left:-width

flex

order进行排序，中间内容flex-grow:1即放大占满，左右内容margin + flex-gorw:0(不放大)，flex-shrink:1(也不缩小)，flex-basis:width(占据固定空间)
